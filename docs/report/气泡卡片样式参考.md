## 七、核心设计特点

### 7.1 毛玻璃效果

- 使用 `backdrop-filter: blur(16px)` 实现毛玻璃效果
- 需要添加 `-webkit-backdrop-filter` 兼容 Safari

### 7.2 弹出动画

- 使用 `cubic-bezier(.34,1.56,.64,1)` 弹性缓动函数
- 从 `scale(.8)` 到 `scale(1)` 配合透明度变化
- 动画时长 0.6s

### 7.3 悬停交互

- 轻微放大 `scale(1.02)`
- 增强阴影深度
- 提升层级 `z-index: 100`

### 7.4 色彩体系

- 用户：粉色调 `rgba(244,114,182,x)`
- 智能体：青绿色调 `rgba(0,212,170,x)`
- 工具：蓝色调 `rgba(96,165,250,x)`
- 背景：深色 `#0a0c10`

### 7.5 发光装饰

- 8px 圆形发光点
- 使用 `box-shadow` 创建发光效果
- 仅用于用户和智能体气泡

## 八、JavaScript 交互逻辑

### 8.1 气泡随机放置（避免重叠）

```javascript
let bubblePositions = [];

function placeBubble(el) {
  const w = window.innerWidth;
  const h = window.innerHeight;
  const elW = Math.min(360, w * .8);
  const elH = el.offsetHeight || 100;
  let x, y, attempts = 0;
  const margin = 80;

  do {
    x = margin + Math.random() * (w - elW - margin * 2);
    y = margin + Math.random() * (h - elH - margin * 2);
    attempts++;
  } while (
    attempts < 50 &&
    bubblePositions.some(p =>
      Math.abs(p.x - x) < elW + 20 &&
      Math.abs(p.y - y) < elH + 20
    )
  );

  bubblePositions.push({ x, y, w: elW, h: elH });
  if (bubblePositions.length > 20) bubblePositions.shift();
  el.style.left = x + 'px';
  el.style.top = y + 'px';
}
```

### 8.2 文本渲染（Markdown 简化）

```javascript
function renderText(text) {
  return text
    .replace(/\*\*(.+?)\*\*/g, '<strong>$1</strong>')
    .replace(/```(\w+)?\n([\s\S]+?)```/g, '<pre><code>$2</code></pre>')
    .replace(/`(.+?)`/g, '<code>$1</code>')
    .replace(/\n/g, '<br>');

```

### 8.3 粒子生成

```javascript
const particles = document.getElementById('particles');
for (let i = 0; i < 30; i++) {
  const p = document.createElement('div');
  p.className = 'particle';
  const size = Math.random() * 6 + 2;
  p.style.width = size + 'px';
  p.style.height = size + 'px';
  p.style.left = Math.random() * 100 + '%';
  p.style.top = Math.random() * 100 + '%';
  p.style.background = `rgba(${Math.random() > .5 ? "244,114,182" : "0,212,170"},${Math.random() * .3 + .1})`;
  p.style.animationDelay = Math.random() * 20 + 's';
  p.style.animationDuration = (15 + Math.random() * 10) + 's';
  particles.appendChild(p);
}
```

## 十、应用建议

### 10.1 与现有项目集成

可将气泡样式应用于：
- `agent_core/static/index.html` - 替换现有日志卡片样式
- `agent_core/static/vue.html` - 升级 Vue 版本界面

### 10.2 样式提取策略

1. **设计令牌**：提取到独立 CSS 变量文件
2. **气泡组件**：封装为可复用 Web Component 或 Vue 组件
3. **动画库**：将关键帧动画提取为独立模块
4. **主题切换**：基于设计令牌实现明暗主题

### 10.3 性能优化

- 粒子数量控制在 30 以内
- 使用 `will-change` 优化动画性能
- 气泡位置缓存避免重复计算
- 使用 `requestAnimationFrame` 调度渲染
